# 🚀 Quick Start

### Clone Repository

```bash
git clone https://github.com/your-username/nexus-ai-os.git
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Backend

```bash
uvicorn main:app --reload
```

### Run Frontend

```bash
npm install
npm run dev
```
